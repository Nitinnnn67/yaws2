Code Of Conduct

The Code of Conduct is a set of guidelines and principles that outline the expected behavior and responsibilities of individuals within the Saket College community. It serves as a framework for maintaining a respectful, inclusive, and safe environment for all students, faculty, and staff.

The Code of Conduct covers various aspects, including academic integrity, respect for diversity, harassment prevention, and responsible use of technology. It emphasizes the importance of ethical behavior, accountability, and mutual respect among all members of the college community.

By adhering to the Code of Conduct, individuals contribute to a positive learning environment that fosters personal growth, collaboration, and academic excellence.

The Code of Conduct is applicable to all students, faculty, and staff at Saket College. It is essential for everyone to familiarize themselves with the code and understand their responsibilities in upholding its principles.
